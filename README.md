## 🐳 Docker

### Docker Build

```bash
docker build . --tag sinamon-backend
```

### Deploy with Docker

1. 수정과 서버에 접속합니다.
2. `deploy-backend.sh` 스크립트를 실행합니다.

## ⚙ 환경변수
