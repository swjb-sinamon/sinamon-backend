## 🚀 시작하기

### 개발 모드
```shell
yarn start
```

### 빌드
```shell
yarn build
```

### 프로덕션
```shell
yarn start:production
```

### ESLint 검사
```shell
yarn lint
```

## 🐳 Docker

### Docker Build

```bash
docker build . --tag sinamon-backend
````

## ⚙ 환경변수

cp .env.example .env
